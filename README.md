# Starter Client Site

Template professionnel de site vitrine réutilisable pour créer des sites modernes pour des entreprises locales : restaurants, artisans, coachs, cabinets, agences, indépendants, associations, etc.

Objectif : construire une base propre, rapide, sécurisée, responsive et facilement personnalisable pour vendre des sites à plusieurs clients.

---

## 1. Stack technique

Ce projet utilise :

- Next.js 16.2.4
- TypeScript
- App Router
- Tailwind CSS v4
- ESLint
- npm
- Dossier `src/`
- AGENTS.md
- CLAUDE.md
- Ollama comme assistant IA local

---

## 2. Objectif du projet

Créer un template de site vitrine premium avec :

- design moderne
- responsive mobile / tablette / desktop
- composants réutilisables
- contenu facile à modifier
- bonne base SEO
- sécurité minimale
- code propre
- déploiement possible sur Vercel
- adaptation rapide à plusieurs entreprises

Exemples de clients possibles :

- restaurant
- coiffeur
- garage
- coach
- avocat
- artisan
- cabinet médical
- association
- agence locale
- freelance
